# Freenance

## Goal of freenance

The goal of Freenance is to empower users to manage their finances locally on their machines, helping them escape the cycle of living paycheck to paycheck. Privacy is the core principle of this application, and as such, all data is stored exclusively on your computer. No information is shared with external servers or third parties, ensuring that your financial data remains secure and private as you work towards financial stability.

## Contributing
We welcome contributions to freenance! If you'd like to contribute, please fork the repository, create a feature branch, and submit a pull request.

#### Steps to Contribute
1. Fork the repository.
2. Create a new branch (git checkout -b feature-name).
3. Make your changes and commit them.
4. Push to your branch (git push origin feature-name).
5. Open a pull request.
6. I'll review the request
7. Congrats you contribute to freenance!
